O Mapa de Falas (Mapeamento de Áudio)

Para usar a voz do Miguel, você precisará gravar os áudios, convertê-los para .mp3 e colocá-los em uma pasta assets/audio.Abaixo, segue o Mapeamento Lógico que você terá que implementar no código mais para frente.

Você pode usar seu Gemini para gerenciar esse banco de dados.
