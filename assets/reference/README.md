# Referências visuais

Os arquivos desta pasta não são os sprites finais do jogo.

Eles foram extraídos dos spritesheets antigos para servir como referência de:

- aparência;
- roupa;
- cabelo;
- paleta;
- proporções;
- conceito visual das ações.

## Atenção

Algumas referências continuam contendo imperfeições herdadas:

- efeitos atravessando limites laterais;
- poeira dividida entre células;
- partes de efeitos vizinhos;
- diferenças de centralização.

Essas imagens não devem ser copiadas diretamente para
`assets/frames`.

Os arquivos finais devem ser gerados novamente seguindo
`docs/CONTRATO_DE_SPRITES.md`.
